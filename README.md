# Mosoro Documentation

[![CI](https://github.com/mosoroio/mosoro-docs/actions/workflows/ci.yml/badge.svg)](https://github.com/mosoroio/mosoro-docs/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Public documentation for the [Mosoro](https://github.com/mosoroio/mosoro-core) platform.

Built with [MkDocs](https://www.mkdocs.org/) and [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

## Local Development

```bash
pip install mkdocs-material
mkdocs serve
```

Open http://localhost:8000 to preview.

## Deployment

Documentation is automatically deployed to GitHub Pages on push to `main`.
